# Changelog

Все заметные изменения DayLane. Формат — по [Keep a Changelog](https://keepachangelog.com/ru/),
версии — по [SemVer](https://semver.org/lang/ru/).

## [1.0.0] — 2026-06-17

Первый релиз v1 (Android, локально, без сети).

### Добавлено
- Главный экран в стиле «Журнал»: дата-герой с навигацией по дням (свайп/стрелки,
  возврат к сегодня), три выделенные сворачиваемые секции **Вчера/Сегодня/Завтра**
  с лентами-заголовками и цветными маркерами.
- Кнопки «+» на секциях Сегодня/Завтра — добавление дела сразу на нужный день.
- Карточка дела: один день / период, всплывающие даты и время, подпункты с
  двусторонней синхронизацией (все подпункты ⇄ дело), примечание.
- Жёсткие каскадные зависимости «Начать после дела» с запретом циклов.
- Перенос невыполненного на сегодня (ручной + опциональный автоперенос).
- Календарь: виды неделя/две недели/месяц, периоды — полосы по дорожкам
  (жадная укладка, лимит дорожек + «+N»), однодневные — точки под числом дня (+N).
- Две задачные краски: однодневные — синий, многодневные — янтарь.
- Локальные напоминания (по умолчанию выключены), светлая и тёмная темы,
  русская локализация.

### Технически
- Flutter + Drift (SQLite) + Riverpod. Доменная логика (scheduling, lanes,
  dependencies, carry_over) покрыта юнит-тестами.
