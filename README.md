# DayLane

Локальное Android-приложение на Flutter для дел с тремя горизонтами времени
(**вчера / сегодня / завтра**) и наглядным календарём, где многодневные дела
показаны полосами по дорожкам. Без сервера, аккаунтов, ИИ и сети — всё хранится
локально.

## Возможности (v1)

- Дата-герой с навигацией по дням и три сворачиваемые секции вчера/сегодня/завтра.
- Дела на один день и периоды; жёсткие каскадные зависимости «Начать после дела».
- Подпункты с синхронизацией со статусом дела, примечания, напоминания.
- Перенос невыполненного на сегодня (вручную или автоматически).
- Календарь: неделя / две недели / месяц; периоды — полосы по дорожкам,
  однодневные — точки; лимит дорожек с «+N».
- Светлая и тёмная темы, русский интерфейс.

## Стек

Flutter · Drift (SQLite) · Riverpod · flutter_local_notifications · intl.

Доменная логика (`lib/domain/`) — чистая, покрыта юнит-тестами.

## Сборка

```bash
flutter pub get
flutter test                 # юнит-тесты доменной логики
flutter run                  # на подключённом Android-устройстве/эмуляторе
flutter build apk --release  # релизный APK
```
